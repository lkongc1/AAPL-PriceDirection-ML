# AAPL-PriceDirection-ML
Modelo de aprendizaje estadístico para predecir la dirección del precio de apertura de acciones del NASDAQ, con un caso práctico aplicado a Apple (AAPL). El proyecto utiliza datos OHLCV extraídos desde yfinance e implementa técnicas avanzadas de Machine Learning para generar señales BUY/SELL/NEUTRAL. Totalmente open source bajo licencia MIT.
